Малко е ранично още за тест, но няма лошо да почнем да споделяме ресурси за подготовката от рано :)

1. W3schools CSS qwiz: http://www.w3schools.com/css/css_quiz.asp

2. Nettuts+ :

2.1. Beginner CSS: http://net.tutsplus.com/articles/quizzes/nettuts-quiz-1-beginner-css/

2.2 CSS specificity and cascading: http://net.tutsplus.com/articles/quizzes/nettuts-quiz-5-css-specificity-and-cascading/

2.3. Basic css3 techniques: http://net.tutsplus.com/articles/nettuts-quiz-10-basic-css3-techniques/

3. Интересно и кратко тестче: http://www.boogiejack.com/CSSquiz.html

4. Някои въпроси се повтарят от W3Schools, но има и някои оригинални:
 http://www.cssportal.com/css-quiz/

5.  CSS тестове, по теми, но за съжаление ( или за радост ? :D ) е на немски език: http://www.thestyleworks.de/quiz/index.shtml

6. Един колега бе намерил, нещо като игричка да изреждане HTML тагове. Познайте какво е това :)  - http://thecssquiz.com/

7. Не е точно тест, но му хвърлете едно око, за забавление :) -- https://developer.mozilla.org/en-US/demos/detail/css-quiz/launch

8. Codeacademy - доста познат сайт, като моето мнение е че някои от по-малките му упражнения са като тестчета - http://www.codecademy.com/tracks/web

9. CSS3 Basic тест: http://tutorialzine.com/quizzes/take/140/

Ако намеря още, ще ги добавя тук.

В скоро време и аз ще пусна една тестче, за CSS-Overview, като мисля да правя по един за всеки урок + тестовете от колегите + тестовете от интернет = ще имаме доста добри ресурси за подготовка и добра основа за изпита :)

Edit: Щях да забравя - към колегата @petromaxa - много як тест ! Едно голямо браво от мен :)
